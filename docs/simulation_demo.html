01 — Why Simulation

業界標準アームをコスト0円で動かす ―― シミュレーション戦略の正当性

⚠ Problem
実機導入の壁
  • UR5e 本体 200〜300万円の初期費用
  • 故障時の修理費・部品交換が高額
  • 安全柵・架台等の周辺設備が必要
  • 運搬・設置・キャリブレーションに数日
  • 失敗を恐れアグレッシブな実験不可
✦ Solution
Gazebo + UR5e フルシム環境
  • UR 公式 URDF を Gazebo に投入（0円）
  • MoveIt 2 で逆運動学・軌道計画
  • カメラ画像を ROS 2 トピックで取得
  • ヘッドレス実行で Mac でも安定動作
  • FFmpeg で MP4 自動生成 → 共有可能
✓ Outcome
得られる検証結果
  • Pick & Place の成功率・所要時間
  • 軌道計画ロジックの動作確認
  • Vision → Manipulation の連携検証
  • プレゼン用の説得力ある動画
  • 実機導入時の即戦力スクリプト資産
▼ ▼ ▼ これを実現する 採用シミュレーション技術 ▼ ▼ ▼

04 — Getting Started

起動コマンドと拡張ロードマップ ―― 1コマンドで動画生成

🔨 シミュレーションイメージビルド
docker compose --profile simulation build。MoveIt 2 + UR description を含む環境を1回構築（初回 10-15分）。
🎬 動画自動生成（メインユース）
./scripts/record_demo.sh 一発でヘッドレス Gazebo 起動 → Pick & Place 実行 → output/demo.mp4 出力まで完結。
🖥 デバッグ用 GUI 起動（Linux のみ）
docker compose --profile simulation up gazebo で GUI 起動、RViz 2 で軌道計画を可視化しながら開発可能。
📊 結果確認
生成された MP4 を任意のプレーヤーで再生、ROS 2 ログは logs/ に保存。CI でも同じ動画が自動生成される。
