01 — Why JetBot

シミュレーションから実機検証へ ―― JetBot を選定する理由

⚠ Problem
シミュレーション単独の限界
  • カメラノイズ・露出変動が再現困難
  • 推論レイテンシの実測値が取れない
  • GPU 発熱・電力制約の影響が不明
  • モーター摩擦・遅延が理想化される
  • 統合系のリアル挙動が掴めない
✦ Solution
JetBot による実機検証
  • Jetson 内蔵 GPU で PyTorch 推論を実機実行
  • 4万円台でビジョン AI 一通り検証可能
  • Differential drive のシンプル運動学
  • Docker でホスト OS と分離開発
  • 本プロジェクトの ROS 2 ノードがほぼ流用
✓ Outcome
実機検証で得る知見
  • 推論レイテンシ・FPS の実数値
  • End-to-End 遅延（カメラ→検出→動作）
  • バッテリー駆動時の連続動作時間
  • 学習済モデルの実機汎化性能
  • ROS 2 ノード単独差替えの実証
▼ ▼ ▼ これを支える JetBot ハードウェア構成 ▼ ▼ ▼
🧠
Jetson Nano
4GB / Maxwell 128コア
  • CUDA / cuDNN / TensorRT 対応
  • JetPack 4.6.1 / Ubuntu 18.04
  • 消費電力 5W / 10W 切替
  • 4GB LPDDR4 共有メモリ
📷
IMX219 Camera
8MP / CSI-2 接続
  • 1280×720 @ 30fps 安定動作
  • CSI 経由で USB より低遅延
  • v4l2_camera で ROS 2 直接対応
  • FOV 約 77° の標準画角
⚙️
Motor Driver
TB6612FNG + PCA9685
  • I2C 経由で PWM 制御
  • 左右独立 DC モーター駆動
  • Differential drive 運動学
  • 最大速度 約 0.5 m/s
🔋
Battery
18650 Li-Ion × 2
  • 3,000mAh / 7.4V 構成
  • 連続動作 30〜60分
  • 保護回路付き必須（安全要件）
  • USB-C 給電と切替可能
Key Takeaway JetBot は 「シミュレーションでは取れない実機メトリクス」を 4〜5万円・GPU 推論付きで検証できる最小構成 2 / 5

04 — Procurement & Next Actions

発注リストと初週アクション ―― 即着手のための具体タスク

🛒 必須品（Phase 1 / 約 4〜5万円）
JetBot AI Kit (Nano 4GB 同梱版) + microSD 64GB + AC アダプタ + 18650 電池。詳細は docs/hardware.md を参照。
🛍 推奨追加品（Phase 2 / 約 1〜2万円）
HDMI モニター + キーボード（初期設定用、SSH 後不要）+ ジャンパーケーブル + 予備モバイルバッテリー。
🏬 推奨購入先
第一候補: Switch Science（日本語サポート・即納）/ 第二候補: Amazon JP（価格優先）。Yahboom 直送は2-4週間掛かるため非推奨。
⚠️ 発注前チェックリスト
Nano 4GB 同梱か / IMX219 カメラか / 18650 保護回路付きか / Wi-Fi モジュール同梱か ―― この4点を商品ページで必ず確認。
