## 発注前チェックリスト

- [ ] JetBot キットに **Jetson Nano 4GB が同梱されているか**（2GB版は性能不足）
- [ ] **JetPack 4.6.1 以降**対応の SD イメージが入手可能か（Waveshare 公式 Wiki にあり）
- [ ] **カメラが IMX219 8MP** か（HD カメラより歪み補正が容易）
- [ ] WiFi アダプタが Linux Ubuntu 18.04 で**ドライバ不要動作する**型番か
- [ ] 18650 電池は**保護回路付き**であるか（短絡対策、安全上必須）
